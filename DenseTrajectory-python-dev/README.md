# DenseTrajectory-python

## ■概要
pythonコーディングの自己研鑽としてDenseTrajectoryをPythonで実装した。<br>
sample_dt.pyがエントリー関数となっており、KTHデータセットのサンプルコードとなっている。<br>
DenseTrajectory/param.pyに各種パラメータを記載している。
